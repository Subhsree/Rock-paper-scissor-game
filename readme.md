# project Title
ROCK PAPER SCISSOR GAME
## overview

welcome to classic ROCK PAPER SCISSOR game build with HTML , CSS and JAVASCRIPT .This interactive web application lets you engage in the timeless hand game against the computer. Test your strategy as you choose between rock , paper , scissor and see if you can beat computer's choice.
### 
- ### features
    - User-friendly interface with responsive design.
    - Real-time gameplay feedback.
    - Randomized computer choices for an unpredictable challenge.

### how to play 

1) Open the (https://subhsree.github.io/Rock-paper-scissor-game/)live demo in your web browser.
2) Select your choice (rock, paper, or scissors) by clicking the corresponding button.
3) Watch the computer's choice and see who wins the round.
4) Repeat and strive for victory!
